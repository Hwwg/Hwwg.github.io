sqlmap学习

get类型：

1.查看是否有注入点：sqlmap -u 网址
2.出数据库名：sqlmap -u 网址 –dbs
3.出数据库的表名：sqlmap -u 网址 -D security –tables
4.出数据库的列名：sqlmap -u 网址 -D security -T users –columns
5.出数据：sqlmap -u -D security -T users -C id,password,username –dump
